ClassCode:EXIT01.3245.LN01
    * 01 : Library Code
    * 32 : Class Code
    * 45 : Function code
    * LN01 : Line Tab. Any string.
    * '.' Is a Separator.

Class Code & Function Code
    * Start from 00 to 99
    * Only allowed 2 chars for Class Code and 2 chars Function code.
    * Alphabet in the code can be used is A,B,C,D,E .. Z .  Small letter is NOT allowed.
    * For a Function code, refer to a sub-method inside a method, start code with 8 or 9 like 91, 92, 9A, 9G, 9X, 81, 82, 8A, 8D.
        Example .. EXIT01.321D, or EXIT01.32M1
